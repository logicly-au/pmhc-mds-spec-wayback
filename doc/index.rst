National Suicide Prevention Trial Evaluation Data Collection Documentation
==========================================================================

.. This INTRODUCTION will only appear in the online HTML version
   (make.sh html) and not in PDFs or any other renderings

.. only:: builder_html

   As the National Suicide Prevention Trial (NSPT) Evaluation Data Collection is an
   extension of the Primary Mental Health Care Minimum Data Set (PMHC MDS), the
   current PMHC MDS Data online documentation is available to be viewed at https://docs.pmhc-mds.com/.

   The main documentation for the NSPT is organised into several sections:

   * :ref:`nspt-docs`
   * :ref:`reporting-docs`
   * :ref:`start-docs`
   * :ref:`data-docs`
   * :ref:`user-docs`
   * :ref:`resources-docs`
   * :ref:`nspt-faqs`

      (make.sh html) and not in PDF itself (or any other renderings)

   .. only:: builder_html

      Download this specification as a :download:`PDF </_static/NSPT.pdf>`.

   .. End of LINK TO THE PDF

   Getting Help
   ------------

   A dedicated NSPT Evaluation Project Team and PMHC Help Desk is available to
   support Primary Health Networks and Provider Organisations implementing the NSPT
   data collection in relation to the PMHC minimum data set. Visit :ref:`getting-help-FAQs` under :ref:`nspt-faqs`.
